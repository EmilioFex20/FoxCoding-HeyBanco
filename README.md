## 🧐 ¿Qué es HAVI?
Mientras que la banca tradicional funciona como un "forense" (te dice en qué gastaste cuando el dinero ya se fue), **HAVI es el copiloto que toma el volante contigo.** Utilizamos un pipeline de IA híbrido que analiza no solo tus números, sino también tu **sentimiento**. Si detectamos estrés financiero o un ritmo de gasto peligroso, HAVI interviene proactivamente para cuidar tu tranquilidad.

### ✨ Funcionalidades Estrella
* **🧠 IA Prescriptiva:** No espera a fin de mes; te avisa en tiempo real si necesitas pausar una categoría de gasto.
* **🎭 Empatía Automatizada:** Análisis de sentimiento (NLP) para detectar frustración y evitar que los usuarios abandonen el banco.
* **💰 Inversión a un clic:** Detecta dinero ocioso y te sugiere movimientos para generar rendimientos inmediatos.

## 🛠️ Stack Tecnológico
Para garantizar la escalabilidad y la seguridad bancaria, HAVI vive en la nube:
* **Core:** Python & AWS Lambda.
* **Cerebro IA:** Amazon Bedrock & Amazon Lex.
* **Datos:** Amazon S3 & Aurora Serverless.
* **Seguridad:** PII Masking (anonimización de datos) y cifrado AES-256.
